## Booking Workflow

1. User enters:

   * Name
   * Email
   * Phone Number
   * Service Required
   * Branch & Department

2. Clicks **Next**

3. Redirected to Payment Page

4. Scans QR Code and completes payment

5. Checks acknowledgement checkbox

6. Clicks **Submit**

7. ODRS Team receives booking details via email

8. Student receives confirmation email automatically
